# Warm-dark, terminal-soul visual identity

The site adopts a "modern site with terminal *soul*" aesthetic rather than a literal
terminal UI: a warm-dark base (`#17130E`), warm off-white text (`#F5EFE6`), an amber
signature (`#F59E0B`) with a burnt-orange accent (`#E8703A`), IBM Plex Mono for
display/labels paired with IBM Plex Sans for body, and purposeful motion with a few
signature terminal moments (typewriter `whoami`, blinking cursor, prompt-style
reveals). This was chosen to give the rebuild real character — the old site was
near-black, colorless, and scattered with tiny inconsistent accents — while staying
readable and impressive to both recruiters and non-technical clients.

## Considered Options

- **Cool AI-tech (slate/indigo + electric accent)** — slick but common in the dev/AI space.
- **Light paper-terminal** — bold and rare, but a riskier bet.
- **Full terminal / IDE metaphor** — maximum character, but gimmick and recruiter-skim usability risk.
- **Maximalist motion** — heavier, slower, and distracting from content.

## Consequences

- Warmth deliberately diverges from the cold-dark dev-portfolio norm; that uniqueness is the point.
- Terminal is flavor (typography, prompt motifs, accents), never a literal navigable terminal.
- The entire design system — color tokens, type scale, components — is built around these choices, so changing the base palette later is non-trivial.
